# segProject-backend

Backend del sistema de gestión de proyectos.

## Tecnologías
- Java
- Spring Boot
- Spring Security
- JWT
- Maven
- SQL Server

## Requisitos
- Java 17+
- Maven

## Configuración
Configurar el archivo `application.properties` con las credenciales de la base de datos.

## Ejecución
```bash
./mvnw spring-boot:run
